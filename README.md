# Jon Daniel Portfolio Website

A modern portfolio landing page created using HTML and CSS Grid Layout.

## 🚀 Features

- Modern UI Design
- CSS Grid Layout
- Custom Shapes using `clip-path`
- Glassmorphism Effect
- Responsive Structure
- Google Fonts Integration
- Remix Icons
- Image Background Sections

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Google Fonts
- Remix Icons

# 🧠 Concepts Practiced

## CSS Grid

```css
display:grid;
grid-template-columns;
grid-template-areas;
gap;
```

## Flexbox

```css
display:flex;
justify-content:center;
align-items:center;
```

## Clip Path

```css
clip-path:polygon();
```

## Glassmorphism

```css
backdrop-filter:blur(10px);
```
